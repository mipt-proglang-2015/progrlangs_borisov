Сравнение фрагментов изображений.

Генерируется случайное изображение. Изображение разрезается на фрагменты с фиксированным размером.
Фрагменты сравниваются между собой, с целью найти наиболее похожий на текущий (такой процесс выполняется в НИРе).

На небольшом количестве фрагментов с большим размером разреза выигрыш небольшой ~ 10-12% (изображение 300x600 размер окна 100).
На дейтсвительно используемых цифрах (изображение 1200х980 размер окна 5) ~ 30-35%
